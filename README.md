# Wazuh-AWS-Threat Detection

## Overview
Deployed Wazuh SIEM on AWS EC2 Ubuntu instance for log monitoring and threat detection.

## Components
- AWS EC2 (Ubuntu)
- Wazuh Manager
- Wazuh Dashboard
- Wazuh Agent

## Tasks Performed
- Installed Wazuh
- Configured OpenSearch
- Created custom rule (severity level 10)
- Tested alerts using SSH brute force simulation
